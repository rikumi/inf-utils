# NyaaCat Infinite Infernal Utils

A utility mod for the NyaaCat Minecraft server, built on Fabric for Minecraft 1.21.11.

## Requirements

- Minecraft 1.21.11
- Fabric Loader >= 0.19.0
- Fabric API
- Cloth Config API (>= 18.0.0)
- Mod Menu (>= 17.0.0, recommended, for in-game config screen)

## Building

```bash
./gradlew build
```

The built mod jar will be in `build/libs/`.
